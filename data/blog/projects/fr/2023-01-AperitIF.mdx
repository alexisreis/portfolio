## Contexte

Dans le cadre d'un projet scolaire, nous devions créer un moteur de recherche
utilisant le web sémantique et DBpedia. Avec mon groupe de 7 étudiants, nous
avons choisi de créer un moteur de recherche de cocktails.

Nous avons utilisé la base de données [DBpedia](https://wiki.dbpedia.org/) pour
récupérer les données nécessaires à l'affichage :

* les **cocktails** (le nom, la description, les ingrédients, la recette, les
images)
* les **alcools** (le nom, la description, l'image, les cocktails qui les
utilisent)

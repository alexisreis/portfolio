Comme projet d'informatique de fin de deuxième année à l'INSA de Lyon,
nous avons dû développer une interface homme-machine permettant à deux personnes de jouer à un jeu d'échecs sur ordinateurs.

Dans une équipe de 5 étudiants, en deuxième année d'école d'ingénieur, nous avons réparti le travail de la manière suivante :
- 2 personnes sur l'interface graphique (graphismes pour l'échiquier, les pièces, les menus, etc.)
- 3 personnes sur le moteur de jeu (les règles, les déplacements, les pièces, etc.)

Nous avons utilisé le langage orienté objet Java et la bibliothèque Swing.

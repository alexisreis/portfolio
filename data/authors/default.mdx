### Fullstack developer @[Liris](https://liris.cnrs.fr)

`Jun 2022 - Aug 2022`

2 months internship at Liris. Conception and development of a web app / tool 
to annotate scanned documents with sensorial data. The documents were from 
the city of Lyon (France) in the 18th century. 
The app, called annotation-app, allow users to : 
1. visualize scanned documents in a web browser
2. annotate scanned pages with sensorial data and transcribe the text
3. save their work and share it with all the users
4. display sensorial statistics about the documents

This was part of the SYSTESENS project at Liris.

Used technologies : Node, React, Python, Flask, Github

You can find more details [here](/blog/2022-08-annotation-app)
